# Magisters-Sokolov
Собрал в один репо файлы по всем проектам для ВКР

GitHub не может сохранить готовую BERT-модель, т.к. она весит почти 350 Мб.
Поэтому, если запустите тетрадь у себя на компьютере, модель будет подгружаться напрямую с HuggingFace и дообучаться заново. Это не занимает много времени (<15 мин при использовании GPU), но это надо иметь в виду.

CSV-файлы хранят предобработанные данные для обучения модели (т.е. уже результат работы кода). Тетрадь запустится и без них, но если положить их в директорию проекта - программа сама их обнаружит и подгрузит, чтобы сэкономить время.
